# What my app does? 

- Searches for golf courses near users inputted error
# Which APIs I used?
- [Overpass API](https://wiki.openstreetmap.org/wiki/Overpass_API) (queries OpenStreetMap data) for course locations
- [Nominatim](https://nominatim.org/release-docs/latest/api/Search/) for geocoding place names to coordinates.
